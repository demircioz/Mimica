# Mimica

This project consists of creating an UX friendly and ergonomic static website about **haptic gloves** for our 1st semester Computer Sciences Web Development module.  
It was designed to offer an optimal UX, with a modern design, smooth animations and intuitive navigation.

This project focuses on **UI/UX design**, visual communication and front-end web development.
